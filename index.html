Welcome to the jFlowLight wiki!

jFlowLight is a SDN-controller client that allows provisioning and control of an SDN deployment. This includes support for flows, statistics. jFlowLight decouples the control from the northbound API provided by the SDN-Controller. Currently it supports OpenDaylight and Floodlight (experimentally).

Latest Release (Stable)

0.8.2

Quick Usage Guide

Below are some examples of the API usage.

Creating the controller

`final Controller controller = ControllerFactory.builder().
            setControllerType("opendaylight").
            setAddress("192.168.1.55").
            setPort(8181).
            setUser("usr").
            setPwd("strong_pwd").
            build();`

Topology operations

Below are some examples of the API usage for the topology's operations.

Getting all nodes

`controller.getAllNode()`

Getting all hosts

`controller.getHosts()`

Statistics operations

Below are some examples of the API usage for retrieve statistical values such as buyte sent/received/drop, packet sent/received/drop, etc.

Getting stats

`controller.getNodeConnection()`

Flow operations

Below are some examples of the API usage for flow's operations.

Getting flows

`controller.getFlow(switchID,tableID)`

Getting a flow

`controller.getFlow(switchID,tableID,entryID)`

Adding a flow

`controller.addFlowEntry(flowname,switchID,priority,tableID,entryID,ipv4Srcs, ipv4Dsts,ipBit,macSrcs,macDsts,outputPort)`

NOTE: some parameters can be null

Deleting a flow

`controller.deleteFlowEntry(switchID,tableID,entryID)`
